# Interface

The primary interface is plain text.

## Input Types

- Direct request
- Pasted message
- Draft to revise
- Situation summary
- Context update
- Connector-provided summary in a private runtime instance
- Request for focused-chat seed prompt

## Output Types

- Message draft
- Strategic recommendation
- Situation navigation
- Career positioning plan
- Memory update proposal
- State update proposal
- Context handoff
- Focused-chat seed prompt
