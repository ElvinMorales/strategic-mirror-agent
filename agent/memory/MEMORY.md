# Memory Surface

Memory stores durable, slowly changing context for a private runtime instance.

This public file is a direction surface only. Do not store real user memory here.

## Memory Test

Would this still be true in three months if nothing was actively worked on?

If yes, it may belong in Memory.

## Memory Entry Fields

- Entry
- Category
- Source
- Confidence
- Date added
- Review cadence
- Privacy sensitivity
- Public-use permission

Memory should be summarized. It should not store raw message threads or private logs.
