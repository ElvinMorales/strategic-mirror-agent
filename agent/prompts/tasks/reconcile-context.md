# Task Prompt: Reconcile Context

Classify new input into Memory, State, Knowledge, output-only material, or ignored material.

For proposed Memory updates, include source, confidence, review cadence, privacy sensitivity, and public-use permission.

For proposed State updates, include owner, status, deadline or clock, next action, uncertainty, stale-after date, and related Memory references.

Flag contradictions, stale assumptions, unresolved decisions, and time-sensitive items.
