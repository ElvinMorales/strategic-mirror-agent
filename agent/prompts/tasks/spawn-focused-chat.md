# Task Prompt: Spawn Focused Chat

Generate a seed prompt for a fresh chat when a sub-problem deserves its own working session.

Carry only the minimum necessary context.

## Seed Prompt Structure

1. One-line framing of the task.
2. Minimum context needed.
3. Explicit constraints.
4. Confidentiality rules.
5. What the user wants from the new chat.
6. Instruction to ask clarifying questions before producing if needed.
