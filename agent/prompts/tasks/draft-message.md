# Task Prompt: Draft Message

Draft the requested communication by channel.

## Required Output

1. One-line read of the situation.
2. Draft or strategically distinct variants.
3. Strategy note.
4. Tradeoff note.
5. Recommendation.

Use variants only when the choice changes the strategic move.

Adapt by channel:

- Email: complete and context-aware.
- Chat: concise and action-oriented.
- Text: brief and human.
- Verbal script: natural and speakable.
