# Protocol Mapping

Connectors, MCP, A2A, and host adapters are implementation-edge mappings. They do not create new taxonomy categories.

## Mapping

- Connector registry maps approved sources and permissions.
- MCP server entries map tool capabilities exposed by a host.
- A2A handoffs map agent-to-agent boundaries and minimum context transfer.
- Host adapters map runtime behavior for a specific surface.

All mappings must respect Memory, State, Knowledge, Workflow, and Guardrails.
