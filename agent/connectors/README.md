# Connectors

Connectors are optional adapter-layer mappings. The core agent works without them.

This public scaffold includes connector policy, registry templates, host adapter docs, protocol mapping notes, and synthetic examples. It does not implement live connectors.

## Tiers

- Tier 0: manual input.
- Tier 1: read-only local files.
- Tier 2: read-only personal productivity sources.
- Tier 3: draft creation.
- Tier 4: external side effects with explicit approval.
- Tier 5: workplace systems in private or internal instances only.

All connector input must pass through classification, reconciliation, and guardrail review.
