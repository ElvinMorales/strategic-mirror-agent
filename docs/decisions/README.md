# Decision Records

Decision records capture architecture and governance decisions that should stay
visible as the public scaffold evolves.

## Records

- [ADR 0001: Connectors as adapter-layer artifacts](0001-connectors-as-adapters.md)
  records why connectors remain adapter-layer artifacts instead of core
  identity, Memory, State, or runtime behavior.

## When To Add A Decision Record

Add a decision record when a change affects architecture, public/private
boundaries, connector posture, artifact taxonomy, governance rules, or other
repo-wide expectations that future contributors should not have to rediscover.
