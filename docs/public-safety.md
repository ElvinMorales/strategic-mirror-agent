# Public Safety

The public repository contains scaffolding only.

## Safe In Public

- Agent direction files
- Prompt templates
- Memory and state templates
- Connector policy templates
- Synthetic examples
- Guardrails
- Output schemas
- Eval rubrics
- Validation scripts

## Keep Private

- Real workplace communications
- Real names of managers, peers, sponsors, clients, or teams
- Real employer-specific workflows
- Private memory and state
- Raw traces
- Connector credentials
- Local environment files
- Screenshots, documents, emails, or meeting notes from real situations

## Review Standard

Before publishing, inspect examples and docs for private specifics. Synthetic names such as Alex, Morgan, Jordan, Taylor, and ExampleCo are acceptable.
